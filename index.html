프로젝트 경험

  • 역할: 전체 시스템 구성 및 회로 설계, 코드 작성, 시각화 구현

  • 카카오 로그인, 실시간 채팅, 관심사 기반 추천코디, 날씨 API 기반 추천 등 다양한 기능 구현

🤖 자율주행 서빙 로봇 (2-2학기 / 개인 프로젝트)
